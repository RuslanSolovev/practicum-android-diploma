## 📱 **О проекте**

**JobFinder** — учебное Android-приложение для поиска вакансий через HeadHunter API. Проект разработан для освоения работы с REST API, современной архитектурой Android-приложений и реализации сложных фильтров поиска.

**Цели проекта**:
- Практика работы с HeadHunter API
- Реализация многоуровневой фильтрации данных
- Освоение Clean Architecture/MVVM
- Создание пользовательского интерфейса с Bottom Navigation

## ✨ **Функциональность**

### 🔍 **Поиск вакансий**
- **Основной поиск** по названию вакансии, компании или ключевым словам
- **Расширенная фильтрация**:
  - Регион поиска (страна и область)
  - Отрасль/сфера деятельности
  - Уровень зарплаты (минимальная и максимальная)
  - Фильтр по указанию зарплаты в вакансии
- **Интеграция с HeadHunter API** для получения актуальных вакансий

### 💾 **Управление избранным**
- Сохранение интересных вакансий
- Просмотр сохраненных вакансий в отдельной вкладке
- Удаление из избранного
- Локальное хранение избранных вакансий

### ⚙️ **Настройки фильтров**
- Сохранение предпочтений фильтрации
- Сброс фильтров к значениям по умолчанию
- Быстрый доступ к часто используемым фильтрам

## 📱 **Навигация**

Приложение использует **Bottom Navigation** с тремя вкладками:

### 1. 🏠 **Главный экран**
- Поисковая строка для ввода запроса
- Кнопка "Фильтры" для настройки параметров поиска
- Список найденных вакансий с основной информацией
- Карточка вакансии содержит:
  - Название вакансии
  - Название компании
  - Зарплата (если указана)
  - Регион
  - Кнопка "В избранное"

### 2. ⭐ **Избранное**
- Список сохраненных вакансий
- Возможность удаления из избранного
- Поиск по сохраненным вакансиям
- Статус актуальности вакансии

### 3. 👥 **Авторы приложения**
- Информация о разработчиках проекта
- Ссылки на GitHub профили
- Описание целей проекта
- Технические детали реализации

## 🛠 **Технологии**

### **Язык и фреймворки**
- **Kotlin** - основной язык разработки
- **Kotlin Coroutines & Flow** - асинхронное программирование
- **Jetpack Compose** (или XML для View System) - UI
- **Navigation Component** - управление навигацией

### **Архитектура**
- **MVVM (Model-View-ViewModel)** - архитектурный паттерн
- **Repository Pattern** - абстракция доступа к данным
- **Clean Architecture** - разделение ответственности
- **Dependency Injection** (Dagger Hilt/Koin) - управление зависимостями

### **Библиотеки**
- **Retrofit 2 + Moshi/Gson** - работа с HeadHunter API
- **Room Database** - локальное хранение избранных вакансий
- **Coil/Glide** - загрузка и кэширование изображений
- **DataStore/SharedPreferences** - хранение настроек фильтров
- **Paging 3** (опционально) - пагинация списка вакансий

### **API**
- **HeadHunter API** ([api.hh.ru](https://api.hh.ru/))
- Получение списка вакансий по параметрам
- Получение детальной информации о вакансиях
- Получение справочников (регионы, отрасли)
